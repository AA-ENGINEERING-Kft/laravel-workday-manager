# Changelog

All notable changes to `laravel-workday-manager` will be documented in this file.

## v1.0.0 - 2025-10-12

### Added
- Initial release
- `MovedWorkday` model for tracking workday modifications
- `WorkdayManager` service with helper methods
- Migration for `moved_workdays` table
- Factory for testing
- Comprehensive test coverage
- Laravel 11 & 12 support
- PHP 8.2, 8.3, 8.4 support
